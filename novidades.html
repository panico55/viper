Dying Light de graça: veja requisitos para rodar no PC

A partir desta quinta-feira (6), o jogo Dying Light: Enhanced Edition ficará disponível gratuitamente na Epic Games Store.
Porém, quem quiser aproveitar essa iniciativa da produtora Techland precisa realizar o download até
o dia 13 de abril — quinta-feira da próxima semana.

Veja também: Dying Light detalha update para nova geração no PlayStation e Xbox

Junto dessa promoção, a produtora recentemente também anunciou que os usuários poderão fazer a linkagem de suas contas da Epic Games com a TechlandGG, na intenção de adquirir um pacote de boas-vindas exclusivo. Inclusive, esse mesmo pacote estará disponível tanto para novos jogadores quanto veteranos.

Requisitos para rodar Dying Light: Enhanced Edition no PC

Abaixo, confira quais são os requisitos mínimos para que o título possa rodar no seu computador!

Requisitos Mínimos

Sistema Operacional: Windows 7 64-bit ou Windows 8 64-bit ou Windows 8.1 64-bit

Processador: Intel Core i5-2500 3.3 GHz ou AMD FX-8320 3.5 GHz

Memória: 4 GB RAM DDR3

Armazenamento: 40 GB de espaço livre

Gráficos: NVIDIA GeForce GTX 560 ou AMD Radeon HD 6870 (1GB VRAM)

DirectX: 11ª Versão

E os requisitos recomendados pelos desenvolvedores são:

Requisitos recomendados

Sistema Operacional: Windows 7 64-bit ou Windows 8 64-bit ou Windows 8.1 64-bit

Processador: Intel Core™ i5-4670K 3.4 GHz ou AMD FX-8350 4.0 GHz

Memória: 8 GB RAM DDR3

Armazenamento: 40 GB de espaço livre

Gráficos: NVIDIA GeForce GTX 780 ou AMD Radeon R9 290 (2GB VRAM)

DirectX: 11ª Versão
